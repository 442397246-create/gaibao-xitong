# 公共导航菜单系统使用说明

## 概述
本系统将所有页面的导航菜单统一管理，现在只需要修改 `导航菜单.html` 中的内容，所有页面的导航都会自动更新。

## 文件说明
- `导航菜单.html` - 导航菜单组件（包含菜单模板、样式、弹窗）
- `导航.js` - 导航菜单加载器（负责渲染导航和公共函数）
- `common.js` - 原有的公共函数文件（保持不变）

## 如何将现有页面改为使用公共导航菜单

### 1. 在 `<head>` 中引入导航组件
在 `</head>` 之前添加：
```html
<script src="导航菜单.html"></script>
<script src="导航.js"></script>
```

### 2. 精简 CSS 样式
删除原来的导航相关样式（保留其他样式），删除：
- `#mainPage` 相关
- `#sidebar` 相关 
- `.logo` 相关
- `nav` 相关
- `.menu-group` 相关
- `.menu-item` 相关
- `.main-content` 相关
- `.header` 相关
- `.user-dropdown` 相关
- `.dropdown-menu` 相关

### 3. 精简 HTML 结构
将原来的：
```html
<div id="mainPage">
    <aside id="sidebar">...</aside>
    <div class="main-content">
        <header class="header">...</header>
        <main class="content">
            <!-- 页面内容 -->
        </main>
    </div>
</div>
```

改为：
```html
<div id="mainPage">
    <main class="content">
        <!-- 页面内容 -->
    </main>
</div>
```

### 4. 删除公共弹窗
删除以下三个弹窗的 HTML：
- `profileModal`（修改个人资料）
- `changePasswordModal`（修改密码）
- `logoutModal`（退出登录）

### 5. 删除公共函数
删除以下函数定义：
- `openProfileModal()`
- `closeProfileModal()`
- `saveProfile()`
- `openChangePasswordModal()`
- `closeChangePasswordModal()`
- `changePassword()`
- `openLogoutModal()`
- `closeLogoutModal()`
- `confirmLogout()`

### 6. 修改初始化代码
在 `DOMContentLoaded` 事件中，在最前面添加：
```javascript
if (window.Navigation && Navigation.init) {
    Navigation.init('页面标题'); // 例如：'母包列表'、'渠道列表' 等
}
```

同时删除以下初始化代码：
- `UI.initSidebar()`
- `document.getElementById('toggleSidebar').addEventListener('click', UI.toggleSidebar)`

## 如何修改导航菜单

### 1. 修改菜单项
编辑 `导航菜单.html` 中的 `window.NavigationTemplate.sidebar()` 函数中的菜单项。

### 2. 修改页面标题
在各页面的 `Navigation.init('页面标题')` 中修改标题。

### 3. 修改样式
编辑 `导航菜单.html` 中的 `window.NavigationTemplate.styles()` 函数中的样式。

查看 `签名列表.html` 作为已改造好的示例。

## 优点
1. **统一维护** - 只需要修改 `导航菜单.html` 一个文件
2. **独立可用** - 每个 HTML 文件独立打开时仍能正常显示
3. **样式一致** - 所有页面使用相同的导航样式
4. **代码精简** - 减少了重复代码
